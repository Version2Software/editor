Editor Plugin Changelog

1.1.0 -- In progress...

  • Added the ability to have multiple documents open at the same time.
  • Added the ability to save and open documents.
  • Added an automatic check on startup to see if a new version of the plugin is available.
  • Removed shared editor button from group chat.
  • Fixed an issue where where an NullPointerException could occur when removing the PacketListeners.

1.0.1 -- April 3, 2007

  • Set the minimum version of Spark to 2.5.0.
  • Reworked the way PacketExtensionFilters are registered so they work with Spark 2.5.0/Smack 3.0.0.
  • Added a Help/About menu.
  • Added a changelog.html file.
  • Added a copy of GNU Lesser Public License (LGPL) to the distribution.

1.0.0 -- March 13, 2007

  • Initial release.
